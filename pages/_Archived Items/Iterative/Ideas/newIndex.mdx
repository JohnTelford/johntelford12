## Overview

Creating websites is difficult, complex, time consuming, and expensive.

Some times companies may have little interest in how their websites are constructed, as long as they have the features and functionality needed to satisfy their target market. The cost of error and security remediation, may cost a magnitude more money than creating the websites.

This website is about optimizing website project life time costs. It describes the construction of the framework used to build this website using Iterative Development and web component technology.
